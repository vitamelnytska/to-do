# ToDo App

## Technologies:

- NestJS
- TypeScript
- MongoDB

## Features:

- add task
- add color of task
- delete task
- edit status of task
- when status is "done" the task will disappear
- edit task
- sort tasks by priority
- sort tasks by duration
