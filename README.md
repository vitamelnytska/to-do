# ToDo App

## Technologies:

- NestJS
- TypeScript
- MongoDB

## Features:

- creating a todo-list
- delete the list
- add task
- add color of task
- delete task
- edit status of task
- when status is "done" tha task will disappear
- edit task
- sort tasks by priority
- sort tasks by duration 

## Running the app

```
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
