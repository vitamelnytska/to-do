# ToDo App

## Technologies:

- NestJS
- TypeScript
- MongoDB

## Features:

- creating a todo-list
- sort tasks by priority
- sort tasks by duration

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
